Платформенная бибилиотека, которая используется в сервисах монорепозитория

сloser - утилита для gracefull shutdown
health - утилита для проверки запуска приложения
kafka - обертка для удобства работы с брокером сообщений
logger - утилита для логирования
migrator - утилита для наката миграций
testconainers - утилита для интеграционного тестирования микросервиса в связке с бд
